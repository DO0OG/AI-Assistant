# 아리 AI 어시스턴트 업그레이드 가이드

## 새로운 기능

### 1. 텍스트 채팅 인터페이스
- **기능**: AI와 텍스트로 대화할 수 있는 GUI 인터페이스
- **사용법**: 
  - 시스템 트레이 아이콘 우클릭 → "텍스트 채팅 열기"
  - 채팅창에서 메시지 입력 후 엔터 또는 전송 버튼 클릭
- **특별 명령어**:
  - `학습: [주제]` - 특정 주제에 대해 인터넷에서 자동 학습
  - `배워: [주제]` - 위와 동일한 기능

### 2. 자동 학습 시스템
- **기능**: 인터넷에서 자동으로 정보를 수집하여 학습
- **학습 소스**:
  - Wikipedia 한국어/영어 페이지
  - RSS 뉴스 피드
  - 사용자가 요청한 특정 주제
- **설정**:
  - 텍스트 채팅 인터페이스에서 "자동 학습 활성화" 체크박스로 제어
  - 기본적으로 1시간마다 자동 학습 실행

## 설치 및 업그레이드

### 새로운 의존성 설치
```bash
pip install beautifulsoup4 selenium newspaper3k feedparser wikipedia
```

또는 requirements.txt를 사용하여:
```bash
pip install -r requirements.txt
```

### spaCy 한국어 모델 설치 (선택사항)
```bash
python -m spacy download ko_core_news_sm
```

## 사용 방법

### 텍스트 채팅 사용하기
1. 아리 프로그램 실행
2. 시스템 트레이에서 아리 아이콘 우클릭
3. "텍스트 채팅 열기" 선택
4. 채팅창에서 메시지 입력

### 자동 학습 활용하기
1. 텍스트 채팅창에서 "자동 학습 활성화" 체크
2. 특정 주제 학습: `학습: 인공지능` 입력
3. 시스템이 자동으로 관련 정보를 수집하여 학습

### 음성 명령으로 학습하기
음성으로도 학습 명령을 사용할 수 있습니다:
- "학습: [주제]를 배워줘"
- "배워: [주제]에 대해 알려줘"

## 주요 특징

### 지능형 응답 시스템
- 사용자 입력에 대한 맥락적 이해
- 학습된 정보를 바탕으로 한 정확한 응답
- 실시간 감정 분석 및 개체 인식

### 자동 모델 저장
- 학습한 내용은 자동으로 `saved_model/` 폴더에 저장
- 프로그램 재시작 시 이전 학습 내용 자동 로드
- 지속적인 개선과 학습

### 멀티모달 인터페이스
- 음성 명령과 텍스트 입력 동시 지원
- 일관된 AI 응답 시스템
- 사용자 친화적인 GUI

## 설정 옵션

### 자동 학습 설정
- **학습 간격**: 기본 1시간 (코드에서 수정 가능)
- **세션당 최대 학습 문서**: 10개
- **학습 주제**: AI, 기술, 과학, IT 등 사전 정의된 주제

### 채팅 인터페이스 설정
- 채팅 기록 자동 저장
- 대화 내용 수동 저장 기능
- 채팅 기록 삭제 기능

## 문제 해결

### 일반적인 문제들

1. **텍스트 채팅창이 열리지 않는 경우**
   - AI 어시스턴트가 제대로 초기화되었는지 확인
   - 로그 파일에서 오류 메시지 확인

2. **자동 학습이 작동하지 않는 경우**
   - 인터넷 연결 상태 확인
   - Wikipedia API 접근 가능 여부 확인

3. **한국어 처리 문제**
   - spaCy 한국어 모델 설치 확인
   - 영어 모델로 폴백되는 것은 정상

### 로그 확인
프로그램 실행 중 발생하는 모든 로그는 `logs/` 폴더에 저장됩니다.
