# 테마 커스터마이징 가이드

## 테마 파일 위치

빌드된 EXE를 한 번 실행하면 기본 테마 JSON이 아래 위치에 준비됩니다.

```text
%AppData%\Ari\theme
```

개발 환경에서는 기본 파일이 아래 폴더에 있습니다.

```text
VoiceCommand/theme
```

## 수정 방법

1. `%AppData%\Ari\theme` 폴더를 엽니다.
2. `default.json`, `sunset.json`, `forest.json` 같은 파일 중 원하는 파일을 수정합니다.
3. 설정창에서 해당 테마를 다시 선택하거나 적용합니다.
4. 열린 UI 위젯이 실시간으로 다시 그려집니다.

## 권장 수정 범위

- 배경색
- 카드/패널 색상
- 강조색
- 텍스트 색상
- 글꼴 이름
- 글꼴 배율

## 주의사항

- JSON 문법이 깨지면 해당 테마는 로드되지 않을 수 있습니다.
- 색상은 가능한 한 `#RRGGBB` 형식을 유지하는 편이 안전합니다.
- 테마 변경은 UI에만 영향이 있고, TTS 워커나 음성 엔진은 재시작되지 않아야 정상입니다.

## 운영 팁

- 먼저 `default.json`을 복사해서 새 이름으로 저장한 뒤 수정하는 방식이 안전합니다.
- 명도가 너무 낮은 배경과 너무 밝은 텍스트를 섞으면 일부 패널 가독성이 떨어질 수 있습니다.
- 실시간 반영이 안 보이면 설정창에서 다시 적용해 즉시 갱신할 수 있습니다.
