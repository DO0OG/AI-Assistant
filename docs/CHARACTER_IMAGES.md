# 캐릭터 애니메이션 이미지 가이드

아리(Ari) 어시스턴트의 캐릭터 위젯은 특정 파일명 규칙을 따르는 PNG 이미지 시퀀스를 사용하여 애니메이션을 구현합니다. 

## 1. 이미지 파일 규칙
- **형식**: 투명 배경을 가진 `.png` 파일
- **파일명**: `{동작이름}{프레임번호}.png` (예: `idle1.png`, `idle2.png` ...)
- **위치**: `VoiceCommand/images/` 폴더

---

## 2. 필수 이미지 리스트 (기본 동작)

| 동작 이름 (Prefix) | 권장 프레임 수 | 설명 |
| :--- | :---: | :--- |
| **idle** | 8 | 가만히 서 있을 때 (기본 상태) |
| **walk** | 9 | 바닥을 걸어 다닐 때 |
| **drag** | 8 | 마우스로 잡고 끌 때 |
| **fall** | 8 | 공중에서 떨어지는 동작 |
| **sit** | 9 | 바닥에 앉아 있을 때 |

---

## 3. 권장 이미지 리스트 (특수 동작)

이 이미지들이 없으면 기본적으로 `idle` 또는 `sit` 상태로 대체됩니다.

| 동작 이름 (Prefix) | 권장 프레임 수 | 용도 |
| :--- | :---: | :--- |
| **surprised** | 2 | 놀람, 화남 감정 표현 시 |
| **sleep** | 4 | 평온, 걱정 감정 또는 밤 시간대 |
| **climb** | 6 | 벽을 타고 올라가는 동작 |
| **ceiling** | 4 | 천장에 붙어 있을 때 |

---

## 4. 감정 표현 시스템 (Emoji + Animation)

아리는 말풍선에 **이모지**를 표시하여 세밀한 감정을 전달합니다. 감정 태그는 설정된 언어에 따라 자동으로 전환됩니다.

| 감정 (ko / en / ja) | 표시 이모지 | 연결 애니메이션 |
| :--- | :---: | :--- |
| **기쁨 / joy / 喜び** | 😊 | walk / idle + 점프 |
| **슬픔 / sad / 悲しみ** | 😭 | sit / sleep |
| **화남 / angry / 怒り** | 💢 | surprised |
| **놀람 / surprised / 驚き** | 😲 | surprised |
| **평온 / calm / 穏やか** | ☕ | idle / sit |
| **수줍 / shy / 恥ずかしい** | ☺️ | sit / idle |
| **기대 / excited / 期待** | ✨ | walk / idle + 점프 |
| **진지 / serious / 真剣** | 🧐 | sit |
| **걱정 / worried / 心配** | 😟 | sit / idle |

---

## 5. 제작 팁
- **방향**: 기본적으로 **오른쪽**을 바라보는 기준입니다. (왼쪽 이동 시 자동 반전)
- **최적화**: 프로그램에서 1.5배 확대하므로 128px~256px 크기가 적당합니다.
